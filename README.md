# Market Watch AI

A next-generation crypto market tracking widget powered by simulated AGI capabilities.

## Features

*   **Real-time Market Data**: Live price updates for BTC, ETH, and other cryptocurrencies.
*   **Advanced Technical Analysis**: Interactive charts with RSI, MACD, and Moving Averages.
*   **AI Market Analyst**: Natural language market summaries explaining trends and signals.
*   **Agent Chat Interface**: Chat with the agent to ask about price, trends, and buy/sell signals.
*   **Smart News Feed**: Real-time market news integration.
*   **Dynamic UI**: Glassmorphism design with dark mode and responsive layout.

## Tech Stack

*   **Frontend**: React, TypeScript, Tailwind CSS
*   **Backend**: Node.js, Express (for API proxying)
*   **Data Sources**: Yahoo Finance API

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the development server (Backend + Frontend):
    ```bash
    npm run dev:server
    ```

3.  Open [http://localhost:5173](http://localhost:5173) in your browser.
